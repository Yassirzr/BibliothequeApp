    Mini-Système de Bibliothèque (C#/.NET)

Cette application console permet de gérer une petite bibliothèque numérique.

    Fonctionnalités

·Ajouter un document (Livre, Magazine, PDF)

·Afficher tous les documents

·Rechercher par mot-clé

·Supprimer par ID

·Sauvegarder dans un fichier texte

·Charger depuis un fichier texte
